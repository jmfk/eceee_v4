Gallery Styles Reference

Complete guide for creating custom gallery templates

Available Template Variables

Top-Level Variables

Variable Type Description
images Array Array of image objects with all image data
imageCount Number Total number of images in gallery
multipleImages Boolean True if more than one image (useful for conditionals)
showCaptions Boolean Whether to display image captions
enableLightbox Boolean Whether lightbox functionality is enabled
lightboxButtonClass String Custom CSS classes for lightbox buttons (close, prev, next)
lightboxCloseIcon String Custom HTML for close button icon (SVG or HTML)
lightboxPrevIcon String Custom HTML for previous button icon (SVG or HTML)
lightboxNextIcon String Custom HTML for next button icon (SVG or HTML)
columns Number Number of grid columns (default: 3, customizable via style variables)

Image Object Properties

Each item in the images array contains:

Property Type Description
index Number Zero-based index (0, 1, 2...)
url String Optimized imgproxy URL (1x size)
srcset String Responsive srcset attribute (e.g., "url1 800w, url2 1600w")
displayWidth Number Calculated display width in pixels (1x)
displayHeight Number Calculated display height in pixels (1x)
altText String Alt text for accessibility (required!)
caption String Image caption text
title String Image title
width Number Original image width
height Number Original image height

Basic Template Example

<div class="gallery">
  <div class="gallery-grid" style="grid-template-columns: repeat({{columns}}, 1fr);">
    {{#images}}
      <div class="gallery-item">
        <img 
          src="{{url}}" 
          srcset="{{srcset}}"
          width="{{display_width}}"
          height="{{display_height}}"
          alt="{{alt_text}}"
          loading="lazy">
        {{#caption}}
          <p class="caption">{{caption}}</p>
        {{/caption}}
      </div>
    {{/images}}
  </div>
</div>

Common Scenarios

1) Simple Grid

<div class="gallery">
  <div class="gallery-grid" style="grid-template-columns: repeat({{columns}}, 1fr);">
    {{#images}}
      <div class="gallery-item" data-index="{{index}}">
        <img src="{{url}}" srcset="{{srcset}}" width="{{display_width}}" height="{{display_height}}" alt="{{alt_text}}" loading="lazy">
        {{#showCaptions}}{{#caption}}<p class="caption">{{caption}}</p>{{/caption}}{{/showCaptions}}
      </div>
    {{/images}}
  </div>
</div>
.gallery-grid { display: grid; gap: 1rem; }
.gallery-item img { width: 100%; height: auto; display: block; }

2) Masonry (CSS Grid)

<div class="masonry">
  <div class="masonry-grid" style="grid-template-columns: repeat({{columns}}, 1fr);">
    {{#images}}
      <div class="masonry-item">
        <img src="{{url}}" width="{{display_width}}" height="{{display_height}}" alt="{{alt_text}}">
      </div>
    {{/images}}
  </div>
</div>
.masonry-grid { display: grid; grid-auto-rows: 8px; gap: 8px; }
.masonry-item { border-radius: 6px; overflow: hidden; }
.masonry-item img { width: 100%; height: auto; display: block; }

3) Caption Overlay

<div class="overlay-gallery">
  <div class="overlay-grid" style="grid-template-columns: repeat({{columns}}, 1fr);">
    {{#images}}
      <div class="overlay-item">
        <img src="{{url}}" alt="{{alt_text}}" loading="lazy">
        {{#showCaptions}}
        <div class="overlay">
          {{#title}}<h4>{{title}}</h4>{{/title}}
          {{#caption}}<p>{{caption}}</p>{{/caption}}
        </div>
        {{/showCaptions}}
      </div>
    {{/images}}
  </div>
</div>
.overlay-grid { display: grid; gap: 1rem; }
.overlay-item { position: relative; overflow: hidden; border-radius: 8px; }
.overlay-item img { width: 100%; height: auto; display: block; transition: transform 0.3s; }
.overlay-item:hover img { transform: scale(1.05); }
.overlay { position: absolute; inset: auto 0 0 0; background: rgba(0,0,0,0.6); color: #fff; padding: 0.75rem; transform: translateY(100%); transition: transform 0.3s; }
.overlay-item:hover .overlay { transform: translateY(0); }

4) Lightbox Ready with Custom Buttons

<div class="lb-gallery">
  <div class="lb-grid" style="grid-template-columns: repeat({{columns}}, 1fr);">
    {{#images}}
      <a class="lb-item"
         data-lightbox
         data-lightbox-group="{{galleryGroup}}"
         data-lightbox-style="{{lightboxStyle}}"
         data-lightbox-src="{{lightboxUrl}}"
         data-lightbox-caption="{{caption}}"
         data-lightbox-alt="{{altText}}"
         {{#lightboxButtonClass}}data-lightbox-button-class="{{lightboxButtonClass}}"{{/lightboxButtonClass}}
         {{#lightboxCloseIcon}}data-lightbox-close-html="{{lightboxCloseIcon}}"{{/lightboxCloseIcon}}
         {{#lightboxPrevIcon}}data-lightbox-prev-html="{{lightboxPrevIcon}}"{{/lightboxPrevIcon}}
         {{#lightboxNextIcon}}data-lightbox-next-html="{{lightboxNextIcon}}"{{/lightboxNextIcon}}
         href="{{lightboxUrl}}">
        <img src="{{url}}" srcset="{{srcset}}" 
             width="{{displayWidth}}" height="{{displayHeight}}"
             alt="{{altText}}" loading="lazy">
      </a>
    {{/images}}
  </div>
</div>
.lb-grid { display: grid; gap: 8px; }
.lb-item { cursor: pointer; }
.lb-item img { width: 100%; height: auto; display: block; border-radius: 6px; transition: opacity 0.2s; }
.lb-item:hover img { opacity: 0.9; }

Imgproxy Configuration

Option Description Example
width Target width in pixels 800
height Target height in pixels 600
max_width Maximum width (prevents upscaling) 1600
max_height Maximum height (prevents upscaling) 1200
resize_type "fit", "fill", "crop", or "force" "fill"
gravity "sm" (smart), "ce" (center), etc. "sm"
Tip: Always set max_width and max_height to prevent images from being upscaled beyond their original size!

Best Practices

  • Always use srcset for responsive images
  • Include width and height attributes to prevent layout shift
  • Use loading="lazy" for better performance
  • Provide meaningful alt_text for accessibility
  • Use grid or flexbox for responsive layouts
  • Test with different numbers of images (1, 3, 6, 12)

Example: Grid with Hover Effect

Template:

<div class="hover-gallery">
  {{#images}}
    <div class="item">
      <img src="{{url}}" srcset="{{srcset}}" 
           width="{{display_width}}" height="{{display_height}}"
           alt="{{alt_text}}">
      <div class="overlay">
        <h3>{{title}}</h3>
        <p>{{caption}}</p>
      </div>
    </div>
  {{/images}}
</div>

CSS:

.hover-gallery {
  display: grid;
  grid-template-columns: repeat({{columns}}, 1fr);
  gap: 1rem;
}
.item {
  position: relative;
  overflow: hidden;
}
.item img {
  width: 100%;
  height: auto;
  transition: transform 0.3s;
}
.item:hover img {
  transform: scale(1.1);
}
.overlay {
  position: absolute;
  bottom: 0;
  left: 0;
  right: 0;
  background: rgba(0,0,0,0.7);
  color: white;
  padding: 1rem;
  transform: translateY(100%);
  transition: transform 0.3s;
}
.item:hover .overlay {
  transform: translateY(0);
}
