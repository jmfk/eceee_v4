Carousel Styles Reference

Complete guide for creating custom carousel templates with Alpine.js

Available Template Variables

Top-Level Variables

Variable Type Description
images Array Array of image objects (same structure as gallery)
imageCount Number Total number of images in carousel
multipleImages Boolean True if more than one image
showCaptions Boolean Whether to display captions
autoPlay Boolean Whether carousel should auto-advance
autoPlayInterval Number Auto-play interval in seconds (default: 3)

Image Object Properties

Each image has the same properties as gallery images: url, srcset, displayWidth, displayHeight, altText, caption, title, etc.

Basic Carousel Template

<div class="carousel" x-data="{ current: 0 }">
  <div class="slides">
    {{#images}}
      <div class="slide" x-show="current === {{index}}">
        <img 
          src="{{url}}" 
          srcset="{{srcset}}"
          width="{{displayWidth}}"
          height="{{displayHeight}}"
          alt="{{altText}}">
        {{#caption}}
          <p class="caption">{{caption}}</p>
        {{/caption}}
      </div>
    {{/images}}
  </div>
  
  {{#multipleImages}}
    <button @click="current = (current - 1 + {{imageCount}}) % {{imageCount}}">
      Previous
    </button>
    <button @click="current = (current + 1) % {{imageCount}}">
      Next
    </button>
  {{/multipleImages}}
</div>

Alpine.js Integration

Carousel styles use Alpine.js for interactivity. Common patterns:

State Management

x-data="{ current: 0, paused: false }"

Show Current Slide

x-show="current === {{index}}"

Navigation Buttons

<button @click="current = (current - 1 + {{imageCount}}) % {{imageCount}}">
  Previous
</button>
<button @click="current = (current + 1) % {{imageCount}}">
  Next
</button>

Indicators/Dots

{{#images}}
  <button 
    @click="current = {{index}}"
    :class="{ 'active': current === {{index}} }">
  </button>
{{/images}}

Auto-Play

x-init="setInterval(() => { 
  if (!paused) current = (current + 1) % {{imageCount}} 
}, {{autoPlayInterval}} * 1000)"
Note: Alpine.js interactions won't work in the theme editor preview. Test on an actual page to see carousel behavior.

Example: Modern Carousel with Transitions

Template:

<div class="modern-carousel" 
     x-data="{ current: 0, paused: false }"
     x-init="{{#autoPlay}}setInterval(() => { 
       if (!paused) current = (current + 1) % {{imageCount}} 
     }, {{autoPlayInterval}} * 1000){{/autoPlay}}"
     @mouseenter="paused = true"
     @mouseleave="paused = false">
  
  <div class="slides-container">
    {{#images}}
      <div class="slide" 
           x-show="current === {{index}}"
           x-transition:enter="transition ease-out duration-300"
           x-transition:enter-start="opacity-0 transform scale-95"
           x-transition:enter-end="opacity-100 transform scale-100">
        <img src="{{url}}" srcset="{{srcset}}" 
             width="{{displayWidth}}" height="{{displayHeight}}"
             alt="{{altText}}">
        {{#showCaptions}}
          {{#caption}}
            <div class="caption">{{caption}}</div>
          {{/caption}}
        {{/showCaptions}}
      </div>
    {{/images}}
  </div>
  
  {{#multipleImages}}
    <div class="nav-buttons">
      <button @click="current = (current - 1 + {{imageCount}}) % {{imageCount}}">
        ‹
      </button>
      <button @click="current = (current + 1) % {{imageCount}}">
        ›
      </button>
    </div>
    
    <div class="indicators">
      {{#images}}
        <button 
          @click="current = {{index}}"
          :class="{ 'active': current === {{index}} }">
        </button>
      {{/images}}
    </div>
  {{/multipleImages}}
</div>

Best Practices

  • Use consistent image dimensions for smooth transitions
  • Provide keyboard navigation (arrow keys)
  • Include pause button for accessibility
  • Show progress indicators (dots or thumbnails)
  • Pause on hover for better UX
  • Use object-fit: cover for images
  • Add swipe gestures for touch devices
  • Ensure auto-play respects user preferences
